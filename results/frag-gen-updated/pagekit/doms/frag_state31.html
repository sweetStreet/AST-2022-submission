Highlight

  • Pagekit
  • Version 1.0.0

The Highlight extension is essential for anyone who writes documentation. It provides syntax highlighting for most common languages, like JavaScript, CSS, Markup, PHP and so on. Just enable the extension and it will automatically detect the language and highlight the syntax in preformatted code blocks.

Features

  • Automatic syntax highlighting for your site
  • Support for over 100 languages including CSS, HTML, JavaScript and PHP
